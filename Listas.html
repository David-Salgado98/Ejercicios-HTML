Ejercicio 3

Función:
Un conjunto de instrucciones que realiza una tarea o calcula un valor.
Let:
La instrucción let declara una variable de alcance local con ámbito de bloque(blockscope), la cual, opcionalmente, puede ser inicializada con algún valor.
Const
Esta declaración crea una constante cuyo alcance puede ser global o local para el bloque en el que se declara.
Etiqueta Style:
Es el elemento encargado de indicar la información de estilo.
Etiqueta Nav:
Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación.
Etiqueta Section:
Representa una sección genérica de un documento.
